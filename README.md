# Projeto Refund

Aplicação de gerenciamento e controle de reembolsos desenvolvida de forma prática através das aulas da plataforma Rocketseat. O sistema permite que o usuário registre despesas informando o nome do item, a categoria e o valor correspondente, centralizando os gastos de forma simples e intuitiva.

## Funcionalidades

*   **Cadastro de Despesas:** Entrada de dados otimizada através de campos específicos para o nome da despesa, seleção de categoria e inserção do valor.
*   **Listagem em Tempo Real:** Os itens são adicionados dinamicamente na tela assim que o formulário é enviado.
*   **Cálculo Automático:** O sistema realiza a soma total dos valores e atualiza o contador de despesas instantaneamente a cada nova inserção ou remoção.

## Tecnologias Utilizadas

*   **Front-end:** HTML5, CSS3 e JavaScript (ES6+)
